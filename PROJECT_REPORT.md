# تقرير مشروع تطبيق الويب (Web App Project Report)

## 1. الفكرة والهدف (Idea and Goal)

**الفكرة:**
التطبيق عبارة عن منصة تواصل آمنة ومجهولة الهوية (Anonymous Chat) تتيح للمستخدمين التحدث بحرية والتعبير عن مشاعرهم أو مشاكلهم دون الكشف عن هويتهم. ينقسم المستخدمون إلى دورين رئيسيين:
- **المُعترف (Confessor):** الشخص الذي يرغب في التحدث ومشاركة ما بداخله.
- **الحارس/المستمع (Guardian):** الشخص الذي يقدم الدعم النفسي ويستمع للمُعترف.

**الهدف:**
توفير بيئة آمنة وداعمة للصحة النفسية والتنفيس العاطفي، حيث يمكن للأشخاص التحدث بحرية تامة دون خوف من الأحكام المسبقة. يهدف التطبيق إلى تعزيز التعاطف والدعم المتبادل بين المستخدمين من خلال نظام محادثة نصية وصوتية متطور، مع الحفاظ على الخصوصية التامة.

---

## 2. تقنيات البرمجة والتطوير (Technologies Used)

تم بناء التطبيق باستخدام أحدث تقنيات تطوير الويب لضمان أداء عالٍ وتجربة مستخدم سلسة:

### **الواجهة الأمامية (Frontend):**
- **React.js (v18+):** مكتبة جافاسكريبت الأساسية لبناء واجهات المستخدم التفاعلية.
- **TypeScript:** لغة برمجة تضيف ميزة الأنواع (Types) إلى جافاسكريبت، مما يقلل من الأخطاء ويسهل صيانة الكود.
- **Vite:** أداة بناء (Build Tool) سريعة جداً لتطوير تطبيقات الويب الحديثة.
- **Tailwind CSS:** إطار عمل CSS يعتمد على الفئات (Utility-first) لتصميم واجهات جذابة ومتجاوبة بسرعة.
- **Framer Motion:** مكتبة لإضافة حركات (Animations) وانتقالات سلسة بين واجهات التطبيق.
- **Lucide React:** مكتبة أيقونات حديثة وخفيفة الوزن.

### **الخلفية وقواعد البيانات (Backend & Database):**
- **Firebase Firestore:** قاعدة بيانات NoSQL سحابية في الوقت الفعلي (Real-time) لتخزين الرسائل، طوابير الانتظار، وبيانات المستخدمين.
- **Firebase Authentication:** نظام مصادقة آمن لتسجيل دخول المستخدمين (عبر Google).
- **WebRTC:** تقنية اتصال في الوقت الفعلي تُستخدم لتوفير ميزة المكالمات الصوتية (Audio Calls) بين المستخدمين من متصفح لمتصفح (Peer-to-Peer).

### **إدارة الحالة (State Management):**
- استخدام خطافات React الأساسية (`useState`, `useEffect`, `useRef`) لإدارة حالة التطبيق المعقدة مثل الرسائل، حالة الاتصال، والمكالمات.

## 3. الميزات الأساسية المنجزة (Key Features)

بناءً على طلبات التطوير، تم تضمين الميزات التالية في التطبيق:
1. **نظام الطابور والمطابقة (Queue & Matchmaking):** ربط المستخدمين (Confessor و Guardian) بناءً على اللغة المختارة.
2. **المحادثة الفورية (Real-time Chat):** إرسال واستقبال الرسائل لحظياً.
3. **المكالمات الصوتية (Audio Calls):** اتصال صوتي آمن ومجهول بين الطرفين.
4. **تعديل وحذف الرسائل:** إمكانية تعديل الرسائل خلال 5 دقائق من إرسالها، وحذفها مع ظهور علامة "تم حذف الرسالة" (تم تنفيذها).
5. **الطوابع الزمنية (Timestamps):** عرض وقت إرسال كل رسالة بوضوح (تم تنفيذها).
6. **التفاعلات (Reactions):** إمكانية التفاعل مع الرسائل باستخدام الرموز التعبيرية (Emojis) (تم تنفيذها).
7. **الرد على الرسائل (Reply):** إمكانية الرد على رسالة محددة داخل المحادثة.
8. **مؤشرات الكتابة والقراءة (Typing & Read Receipts):** معرفة متى يكتب الطرف الآخر ومتى قرأ الرسالة.
9. **نظام التقييم والمكافآت (Rating & Rewards):** تقييم المحادثة بعد انتهائها ومنح نقاط (Points) للمستخدمين.

## 4. مخطط توضع الملفات (File Structure)

هيكل المشروع منظم لضمان سهولة الوصول والتطوير:

```text
/
├── index.html                 # نقطة الدخول الرئيسية لصفحة الويب
├── package.json               # ملف إدارة الحزم والاعتمادات (Dependencies)
├── vite.config.ts             # إعدادات أداة البناء Vite
├── tailwind.config.js         # إعدادات وتخصيص Tailwind CSS
├── firebase-applet-config.json # إعدادات الاتصال بمشروع Firebase
├── firebase-blueprint.json    # مخطط قاعدة بيانات Firestore (Schema)
├── firestore.rules            # قواعد الأمان لقاعدة بيانات Firestore
├── src/                       # مجلد الشيفرة المصدرية (Source Code)
│   ├── main.tsx               # ملف الإقلاع الرئيسي لتطبيق React
│   ├── App.tsx                # المكون الرئيسي للتطبيق (يحتوي على منطق المحادثة والمطابقة)
│   ├── index.css              # ملف التنسيقات العامة والمتغيرات (CSS Variables)
│   ├── firebase.ts            # ملف تهيئة اتصال Firebase
│   ├── components/            # مجلد المكونات القابلة لإعادة الاستخدام
│   │   ├── AudioCall.tsx      # مكون المكالمات الصوتية (WebRTC)
│   │   ├── AuthScreen.tsx     # مكون شاشة تسجيل الدخول
│   │   └── AccountDashboard.tsx # مكون لوحة تحكم حساب المستخدم
```

### شرح مبسط للملفات الرئيسية:
- **`App.tsx`:** هو القلب النابض للتطبيق، يحتوي على شاشات اختيار الدور، اختيار اللغة، شاشة الانتظار، واجهة المحادثة بكل ميزاتها (الرسائل، التفاعلات، الحذف، التعديل)، وشاشة التقييم.
- **`AudioCall.tsx`:** يتعامل مع تعقيدات الاتصال الصوتي باستخدام WebRTC وتبادل الإشارات (Signaling) عبر Firestore.
- **`index.css`:** يحتوي على متغيرات الألوان (CSS Variables) التي تسمح بتغيير مظهر التطبيق بسهولة (Theming).
