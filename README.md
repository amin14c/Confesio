## 📖 About The Project

**Confesio** is built on the concept of "Folk Psychological Therapy". It provides a safe haven for individuals to express their thoughts, fears, and confessions anonymously without the fear of judgment. 

Users can enter the platform as either a **Confessor** (someone seeking a listening ear) or a **Guardian** (someone offering support and empathy). The platform pairs them dynamically and facilitates secure text and audio communication, ensuring privacy at every step.

### ✨ Key Features

- 🎭 **Role-Based Anonymous Matchmaking:** Automatically pairs 'Confessors' with 'Guardians' based on their preferred communication language.
- 💬 **Real-Time Encrypted Chat:** Instant messaging synced seamlessly via Firestore, featuring real-time typing indicators and interactive emoji reactions.
- 🎙️ **P2P Audio Calling:** Secure, low-latency voice communication powered by WebRTC.
- 🎙️ **Voice-to-Text Dictation:** Built-in speech recognition to instantly transcribe spoken words into the chat input.
- 🛡️ **Zero-Trust Security & Moderation:** Bulletproof Firestore Security Rules prevent unauthorized access. An advanced Admin Command Center allows dynamic reviewing of active reports, chat logs decryption, and managing user bans.
- ♿ **WCAG Accessible & Responsive UI:** Fully keyboard navigable interface optimized for screen readers, wrapped in a beautifully crafted, immersive Dark/Light/Forest/Midnight theme engine (complete with animated Floating Candles).
